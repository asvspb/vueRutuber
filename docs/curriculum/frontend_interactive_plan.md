# Расширенный интерактивный курс Frontend-разработки (60 уроков)

Этот курс предназначен для глубокого погружения в основы веб-разработки. Программа разбита на три больших модуля по 20 уроков в каждом. Каждый урок включает теоретический минимум и обязательное практическое задание.

---

## Модуль 1: HTML — Структура и Семантика (20 уроков)

**Цель:** Овладеть языком разметки на профессиональном уровне, уделяя особое внимание семантике и доступности.

### Блок 1: Фундамент (Уроки 1-5)
1.  **Введение в Web:** Как работает интернет, роль браузера, структура HTML-документа (`<!DOCTYPE>`, `html`, `head`, `body`).
    *   *Задание:* Создать первую страницу "Hello World" и открыть её в браузере.
2.  **Текстовые теги:** Заголовки (`h1`-`h6`), параграфы (`p`), переносы (`br`), разделители (`hr`).
    *   *Задание:* Сверстать структуру статьи с заголовками разных уровней.
3.  **Форматирование текста:** Жирный (`strong`, `b`), курсив (`em`, `i`), подчеркивание, зачеркивание. Семантическая разница между тегами.
    *   *Задание:* "Исправить" текст, расставив смысловые акценты правильными тегами.
4.  **Списки:** Маркированные (`ul`), нумерованные (`ol`), списки определений (`dl`). Вложенные списки.
    *   *Задание:* Сверстать рецепт блюда (ингредиенты и шаги приготовления).
5.  **Ссылки и якоря:** Тег `<a>`, атрибут `href`, абсолютные и относительные пути, ссылки внутри страницы (`id`).
    *   *Задание:* Создать простое оглавление для длинной статьи с навигацией по якорям.

### Блок 2: Медиа и Контент (Уроки 6-10)
6.  **Изображения:** Тег `<img>`, атрибуты `src`, `alt`, `width`, `height`. Форматы (JPG, PNG, SVG, WebP).
    *   *Задание:* Создать фотогалерею с подписями (используя `figure` и `figcaption`).
7.  **Пути к файлам:** Структура папок проекта. Относительные пути (`./`, `../`).
    *   *Задание:* Подключить изображения из разных папок проекта.
8.  **Аудио и Видео:** Теги `<audio>`, `<video>`, атрибуты `controls`, `autoplay`, `loop`. Поддержка форматов.
    *   *Задание:* Встроить видеоплеер с несколькими источниками (`<source>`).
9.  **Iframe:** Встраивание стороннего контента (YouTube, карты). Безопасность (`sandbox`).
    *   *Задание:* Встроить карту Google Maps и видео с YouTube на одну страницу.
10. **Символы и сущности:** Спецсимволы (`&nbsp;`, `&copy;`, `<`). Эмодзи.
    *   *Задание:* Написать формулу или код на странице, используя HTML-сущности.

### Блок 3: Таблицы и Данные (Уроки 11-15)
11. **Основы таблиц:** `<table>`, `tr`, `td`, `th`.
    *   *Задание:* Сверстать простую таблицу расписания занятий.
12. **Сложные таблицы:** Объединение ячеек (`rowspan`, `colspan`).
    *   *Задание:* Сверстать сложный прайс-лист с объединенными ячейками.
13. **Структура таблицы:** `thead`, `tbody`, `tfoot`, `caption`.
    *   *Задание:* Рефакторинг таблицы расписания с добавлением семантических секций.
14. **Введение в формы:** Тег `<form>`, атрибуты `action`, `method` (GET/POST).
    *   *Задание:* Создать каркас формы поиска.
15. **Базовые поля ввода:** `<input>` (text, password, checkbox, radio). `placeholder`, `value`.
    *   *Задание:* Сверстать форму входа (логин/пароль) и "запомнить меня".

### Блок 4: Продвинутые формы и Семантика (Уроки 16-20)
16. **Расширенные поля ввода:** HTML5 типы (email, tel, number, date, color, range).
    *   *Задание:* Создать форму заказа пиццы с выбором даты и количества.
17. **Элементы управления:** `<select>`, `<option>`, `<textarea>`, `<button>`. Группировка `<optgroup>`.
    *   *Задание:* Добавить в форму заказа выбор города и поле для комментария.
18. **Структура формы:** `<label>` (связь через `for`), `<fieldset>`, `<legend>`.
    *   *Задание:* Сделать форму доступной, связав все лейблы с инпутами.
19. **Семантические теги:** `header`, `nav`, `main`, `article`, `section`, `aside`, `footer`.
    *   *Задание:* Переверстать макет блога, заменив все `div` на семантические теги.
20. **Метаданные и SEO:** Тег `<head>` подробно. `title`, `meta description`, `keywords`, Open Graph.
    *   *Задание:* Подготовить страницу к публикации в соцсетях (настроить превью).

## Модуль 2: CSS — Стилизация и Дизайн (20 уроков)

**Цель:** Научиться создавать визуально привлекательные, адаптивные интерфейсы.

### Блок 1: Основы стилизации (Уроки 1-5)
1.  **Синтаксис и подключение:** Селекторы, свойства, значения. Inline, internal, external стили.
    *   *Задание:* Подключить стили тремя способами и проверить приоритеты.
2.  **Базовые селекторы:** По тегу, классу (`.`), ID (`#`), универсальный (`*`).
    *   *Задание:* Раскрасить разные элементы страницы, используя разные типы селекторов.
3.  **Работа с цветом:** HEX, RGB, RGBA, HSL. Прозрачность (`opacity` vs alpha-channel).
    *   *Задание:* Создать палитру цветов для сайта.
4.  **Работа с текстом:** `font-family`, `font-size`, `font-weight`, `font-style`, `text-align`.
    *   *Задание:* Оформить типографику статьи (заголовки, основной текст, цитаты).
5.  **Декорирование текста:** `text-decoration`, `text-transform`, `line-height`, `letter-spacing`.
    *   *Задание:* Стилизовать ссылки (убрать подчеркивание, добавить при наведении).

### Блок 2: Блочная модель и Фоны (Уроки 6-10)
6.  **Блочная модель (Box Model):** Content, Padding, Border, Margin. Схлопывание отступов.
    *   *Задание:* Визуализировать блочную модель, задав разные цвета для границ и фонов.
7.  **Размеры и Box-sizing:** `width`, `height`, `max/min-width`. Проблема `content-box` и решение `border-box`.
    *   *Задание:* Построить сетку карточек, которые не ломаются при добавлении паддингов.
8.  **Фоны:** `background-color`, `background-image`, `url()`, `repeat`, `position`, `size` (cover/contain).
    *   *Задание:* Создать герой-секцию (hero section) с фоновым изображением на весь экран.
9.  **Градиенты:** Линейные (`linear-gradient`) и радиальные (`radial-gradient`).
    *   *Задание:* Создать кнопку с градиентным фоном.
10. **Границы и Тени:** `border-radius` (круг, овал), `box-shadow`, `text-shadow`.
    *   *Задание:* Сделать карточку товара с тенью и скругленными углами ("Material Design").

### Блок 3: Позиционирование и Flexbox (Уроки 11-15)
11. **Потоки документа:** Блочные (`block`) и строчные (`inline`, `inline-block`) элементы. Свойство `display`.
    *   *Задание:* Сверстать горизонтальное меню, используя `inline-block`.
12. **Позиционирование:** `static`, `relative`, `absolute`, `fixed`, `sticky`. `z-index`.
    *   *Задание:* Разместить иконку "New" поверх карточки товара (absolute внутри relative).
13. **Flexbox: Контейнер:** `display: flex`, `flex-direction`, `flex-wrap`, `justify-content`, `align-items`.
    *   *Задание:* Выровнять элементы меню по центру и разнести их по краям.
14. **Flexbox: Элементы:** `flex-grow`, `flex-shrink`, `flex-basis`, `order`, `align-self`.
    *   *Задание:* Сверстать "резиновую" раскладку (sidebar + content), где контент занимает все свободное место.
15. **Псевдоклассы и псевдоэлементы:** `:hover`, `:active`, `:focus`, `:first-child`, `::before`, `::after`.
    *   *Задание:* Сделать кастомные маркеры списка и тултипы на CSS.

### Блок 4: Grid, Адаптив и Анимации (Уроки 16-20)
16. **CSS Grid: Основы:** `display: grid`, `grid-template-columns/rows`, `gap`. Единица `fr`.
    *   *Задание:* Сверстать макет страницы (header, sidebar, main, footer) на гридах.
17. **CSS Grid: Области:** `grid-template-areas`, именованные линии.
    *   *Задание:* Переверстать макет, используя ASCII-art синтаксис grid-areas.
18. **Адаптивность:** Media queries (`@media`), breakpoints. Mobile-first подход.
    *   *Задание:* Сделать страницу, которая меняет раскладку с 3 колонок на 1 при уменьшении экрана.
19. **Трансформации и Переходы:** `transform` (rotate, scale, translate), `transition`.
    *   *Задание:* Анимировать кнопку при наведении (плавное увеличение и изменение цвета).
20. **Анимации (Keyframes):** `@keyframes`, `animation-name`, `duration`, `timing-function`.
    *   *Задание:* Создать бесконечную анимацию спиннера загрузки.

## Модуль 3: JavaScript — Программирование и Интерактивность (20 уроков)

**Цель:** Научиться писать логику, работать с данными и взаимодействовать с пользователем.

### Блок 1: Основы языка (Уроки 1-5)
1.  **Введение в JS:** Подключение скрипта, консоль, комментарии. Инструкции и точка с запятой.
    *   *Задание:* Вывести в консоль приветствие и результат математической операции.
2.  **Переменные:** `let`, `const`. Именование переменных. Устаревший `var`.
    *   *Задание:* Создать переменные для хранения данных пользователя и вывести их в предложении.
3.  **Типы данных:** String, Number, Boolean, Null, Undefined. `typeof`. Преобразование типов.
    *   *Задание:* Написать скрипт, который преобразует строковые данные из промпта в числа и складывает их.
4.  **Операторы:** Арифметические, присваивания, сравнения (`==` vs `===`), логические (`&&`, `||`, `!`).
    *   *Задание:* Проверить, попадает ли число в заданный диапазон.
5.  **Условные операторы:** `if`, `else if`, `else`. Тернарный оператор.
    *   *Задание:* Написать скрипт "Фейсконтроль" (проверка возраста).

### Блок 2: Циклы и Функции (Уроки 6-10)
6.  **Циклы:** `while`, `for`. Прерывание `break`, `continue`.
    *   *Задание:* Вывести в консоль все четные числа от 1 до 100.
7.  **Функции (Declaration & Expression):** Объявление, параметры, аргументы, `return`.
    *   *Задание:* Написать функцию расчета площади прямоугольника.
8.  **Стрелочные функции:** Синтаксис ES6, отличия от обычных.
    *   *Задание:* Переписать функции из предыдущего урока на стрелочные.
9.  **Области видимости:** Глобальная, блочная, функциональная. Замыкания (базово).
    *   *Задание:* Исправить ошибку доступа к переменной внутри блока `if`.
10. **Работа со строками:** Методы `length`, `toUpperCase`, `indexOf`, `slice`, шаблонные строки (backticks).
    *   *Задание:* Написать функцию "Капитализация" (делает первую букву заглавной).

### Блок 3: Структуры данных (Уроки 11-15)
11. **Массивы:** Создание, доступ по индексу, `length`. Методы `push`, `pop`, `shift`, `unshift`.
    *   *Задание:* Создать список покупок и реализовать функции добавления/удаления товаров.
12. **Перебор массивов:** `for..of`, методы `forEach`, `map`, `filter`.
    *   *Задание:* Из массива цен получить новый массив цен со скидкой.
13. **Объекты:** Свойства, методы, `this`. Доступ через точку и скобки.
    *   *Задание:* Создать объект "Пользователь" с методами "Приветствие" и "Смена имени".
14. **Объекты и Массивы:** Массив объектов. Поиск в массиве объектов (`find`).
    *   *Задание:* Найти в массиве пользователей человека с определенным ID.
15. **Деструктуризация и Spread:** Распаковка массивов и объектов. Оператор `...`.
    *   *Задание:* Объединить два массива и два объекта в новые структуры.

### Блок 4: DOM и События (Уроки 16-20)
16. **DOM-дерево:** Навигация по DOM (`parentNode`, `children`). Поиск элементов (`querySelector`, `getElementById`).
    *   *Задание:* Найти все ссылки на странице и вывести их адреса в консоль.
17. **Манипуляции с DOM:** Изменение содержимого (`innerHTML`, `textContent`), стилей (`style`), классов (`classList`).
    *   *Задание:* Сделать "Ночной режим" (переключение класса `dark-theme` на `body`).
18. **Создание элементов:** `createElement`, `appendChild`, `remove`.
    *   *Задание:* Реализовать добавление элементов в список дел (To-Do List) через JS.
19. **События (Events):** `onclick` vs `addEventListener`. Объект события `event`. Отмена действия браузера (`preventDefault`).
    *   *Задание:* Сделать валидацию формы перед отправкой (проверка на пустоту).
20. **Всплытие и Делегирование:** Как работают события. Делегирование событий на родителя.
    *   *Задание:* Повесить один обработчик клика на весь список, чтобы обрабатывать клики по любым элементам списка (даже новым).

## Финальный проект курса

**"Личный дашборд разработчика"**

Объедините все знания для создания одностраничного приложения (SPA), которое включает:
1.  **Виджет погоды:** (Верстка + JS запрос к API или мок-данные).
2.  **Список задач (To-Do):** С добавлением, удалением и сохранением в LocalStorage.
3.  **Калькулятор:** Полноценный интерфейс на Grid/Flexbox + логика на JS.
4.  **Смена тем:** Светлая/Темная тема (CSS переменные + JS).

**Требования:**
*   Семантический HTML5.
*   Адаптивный CSS (Flex/Grid, Media Queries).
*   Чистый JS (ES6+), разбитый на функции.
